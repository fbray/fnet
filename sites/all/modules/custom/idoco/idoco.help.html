Imports HTML content and assets and presents them in a Drupal context

  • Strips out all but the main 'body' content
  • Removes or replaces legacy tags and attributes
  • Alters file references to work in an Idoco context
  • Runs content through tidy to ensure validity
  • Generates a dynamic, jquery navigation menu
  • Utilizes caching of all generated content if caching is enabled

Idoco needs to do a lot of HTML parsing, particularly for the navigation element. Idoco will take advantage of caching when it is enabled. It takes up to 5 seconds longer to generate a non-cached page, so enabling caching is highly recommended.
